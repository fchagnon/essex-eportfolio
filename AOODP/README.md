# AOODP: Advanced Object-Oriented Design and Programming

MSc Cyber Security, University of Essex Online. This module covers
advanced object-oriented design principles, design patterns, secure
coding practices, software architecture, concurrency, and test-driven
development, culminating in an End of Module Assignment (e-Portfolio and
reflective Professional Development Plan).

## Highlights

**[Unit 9: ShopEase](Unit%209)** — the module's most substantial design
artefact, a layered e-commerce architecture demonstrating Repository,
Abstract Factory, Observer, and Visitor, with every mechanism traceable
to a single underlying principle: decoupling. Includes a full design
rationale document and UML class diagram.

**[Unit 6: Thread-Safe Banking System](Unit%206)** — the module's sole
summative coding exercise, graded 70% (Distinction), covering deadlock
prevention via resource ordering and a full concurrency-aware test suite.

**[Unit 10: TDD E-Learning Platform](Unit%2010)** — built test-first,
using Python's `Protocol` for structural typing and the Decorator
pattern for cross-cutting concerns like audit logging.
